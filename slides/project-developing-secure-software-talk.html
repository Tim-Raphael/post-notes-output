                                        Advisory Databases (RustSec)                                        
                                                                                                                                        
                                                                                                                                        
                                                                                                                                        
                              Public, community-maintained advisory DB on GitHub                                                       
                              Advisories as Markdown + TOML                                                                            
                              Exported to OSV -> GitHub Advisory DB -> Dependabot                                                      
                              Tracking:                                                                                                
                                 Security vulnerabilities (RCE, XSS/CSRF, injection, crypto, covert channels)                          
                                 Soundness issues (safe Rust causing undefined behavior)                                               
                                 Informational advisories (e.g., unmaintained crates)                                                  
                                                                                                                                        
                                                                                                                                        
           Source: RustSec Advisory Database (https://rustsec.org/)
